# Wiki

A multi-domain knowledge project by [@hazl](https://)

### 1.0 - An introduction to this project

Welcome to our **"wiki"** repository! This is essentially where i share and publish articles for our [website]() aswell as the place I share knowledge and writings, on things that I find meaningful.

This project is at it's core, a local folder of my writings, which i plan on making collaborative eventually. For now, it can be accessed as...
- A github repository
- Local folder
- Obsidian vault
- Visual Studio Code project

This GitHub repository is serving a remote mirror of our **"wiki"** articles. These articles are written in plain-text markdown, becuase they are designed to be accessible through multiple domains. [^1]

## What's the personal function of this wiki?
This wiki serves as a public system to share my personal writings, ideas, discoveries and stories. I spend most of my time writing, and this project helps me integrate my private materials, as a public integration ecospace. **Also, the stuff we're exploring in this wiki, is pretty epic and badass!**

## Why isn't this repository included in website?
These articles were not included in the original repository of the site because we wanted to emphasize the importance of the wiki part of the website. The source code and writings should fundamentally exist in separate domains of operation.

Repository contains a publically accessible remote mirror of my articles, drafts, stories, media and writings, for which will be published on our site, https://yvrs.ct.ws
