How is it possible for human beings to work with non-linear systems?

Principle 1: We cannot directly understand, or interact with a non-linear system. However, we can intreact with the symbolic, hierarchial  map of properties in a non-linear system.

This shows up everywhere in complex fields.

A database shows a real-world model of interacting with non-linear, 3-dimensional systems, yet, working with 2-dimensional arrays.

- A database defines properties, not side-effects or relationships
- This allows us to work with non-linear systems, despite not being able to fully predict or control their side effects.

Side-effects = Chaos = Aliveness.

Side-effects, unpredictability, is where static rules turn into dynamic simulations. We see this in primitive forms such as [Cellular Automata]() or extremely complex physics engines that model and simulate accurate real-world interactions.
