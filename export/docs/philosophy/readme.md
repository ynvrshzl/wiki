This project is at it's core, a **local wiki of plain-text files.** 

- Any plain-text editor can open these files and make changes. It's decentralized, open-source human-first information web constructed of plain-text markdown files. 
- The [github repository]() is simply a publically accessible source-view of the wiki through github. 
- Our [website]() is another mirror of the wiki. 
- [Obsidian]() can edit the wiki. 
- The wiki is an interconnected system, and itself, is also accessible through interconnected systems. Meta? Meta!

# Philosophical Constraints & Principles
- This system must be accessible from any plain-text enviornment. Information is the source, and the view is the interchangeable.
