1. when are frameworks useful?

i think for fields such as: code, music,vcinema, literature, games, and fields of similar neighborhood, which involve creative structure.

2. when is a framework detrimental? 

hypothesis: when the framework gets in the way (adding friction where fluidity should be, or adding fluidity where friction should be) of actually creating the original work!

3. what is a framework?

to my understanding, a  framework is a set of agreements and assumptions about what is useful for solving problems in a set or class. 

4. how are frameworks created?

from my research, nobody is clairvoyant (hopefully lol). so we don't know what is useful vs what is unnecessary... we sample ideas and see where problems naturally emerge in work (e.g., creating scales in music, or patterns in architecture).

the idea is:
- we don't create frameworks, we co-discover usefulness
