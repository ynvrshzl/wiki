# The Mechanics of Human Learning 

Using Vi as an example, these ideas are cross-transferrable for virtually all creative human domains

1. Grammar
2. Rehearseal
3. Composition


## 1.0 - Grammar
Learning itself begins before the actual activity; we learn concepts about an area, before we can sample the area.

Learning the grammar of an ability. One by one. Segmented, Serialized. For Vi it looks like learning individual keys, but in most creative human fields, we are applying the same idea; learning one-thing-at-a-time.
