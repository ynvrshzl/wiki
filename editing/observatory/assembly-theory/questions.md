In this class, we study these questions:
- How do we put loose things together, meaningfully?
- How do we finish things?
- How can we learn to write stories? 
- How does a person learn to make music? 
- How does someone learn to paint? 
- Fundamentally, how do we human beings learn creative, expressive abilities? 
- How does a person learn music? Do we begin with theory? Or by listening to music? This feels awfully similar to the classical [chicken or egg question.]() So which comes first? 
- How do we practically learn new abilities? 
- Is there a "most" efficient method to learning?
