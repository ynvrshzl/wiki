How do I know what theory is helpful?

a good litmus test for this is "does it actually help me solve the problems i'm facing?" the problems exist, regardless of theory. but is the theory helping, or making the problem, harder to solve?

Theories can be fun to work on paper, but once it's actually meant to be integrated into human life, we need to understand that we only remember small models; this only works if we can actually integrate it, so that means it needs to be compressed.
