# Building a computer from scratch

Building a computer from scratch, is essentially a sandbox playground project. We can best learn how computers work, by building our own pseudo model. 

Regardless of the "extreme" complexity of systems, everything sustainable is always simplifiable to it's constituents, which is a concept from [(cellular automata)](cellular-automata.md)

With a computer, we begin at the lowest level, and learn how simple rules emerge into complex, capable systems. Moreover, we learn how to wield complex systems, that help solve the hardest problems in nature.
