Causes: Why does this happen? 

Hypothesis 1: Interference between channels.

Fundamentally, there is a difference between writing code, and reading code. When we try to both: create code systems and understand how they will be used, we cannot think properly, __because when code becomes unreadable, it becomes a non-linear relationship structure.__ Even the most complex examples of computer programs, such as Physics engines and Animation engines, do not force the human engineer to track every relationship and variable. It is similar to applying keyframes in animation, where we let the technology interpolate between frames, using our constraints and rules.
