# Re-useability
One of the core ideas in programming is the ability to re-use code. This is not only incredibly time-saving, but necessary when constructing complex systems.

However, re-useability itself is an arbitrary property of a unit, there are semantic types of re-useability;

- **Functional:** the ability to re-use pure function.
- **Logical:** the ability to re-use logic
- **Graphical:** the cascading effects, such as in CSS
- **Parametric:** the ability for code to accept arguments, and re-use other types.

Specifically, these are packed into blocks:

- __Arg(x).reuse(a)__
- $Abstract(x).reuse(1)$ ...the most abstract form of reuseabilty, is the grammar block 'reuse' which modifies a stream of code, to become reuseable.
