# The banning of families

This is when families began to be split apart. Father with sons, mothers with daughters. There were strict regulations as to who would get to have children. Parents weren't allowed to see their children if they failed "the preliminary exams." A set of brutal tests, designed to break the human psyche. Only the most intelligent, could read between the lines, and pass these tests. That as the intent. That was the purpose of these exams. To rule out ignorant parents, and stop children from being brought into this world.
