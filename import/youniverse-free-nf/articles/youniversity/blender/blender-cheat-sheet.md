# Blender cheat sheet
The most useful, common shortcuts to remember

| Keys | Description | 
| --- | --- |
| Alt Z | Toggle x-ray mode |
| Z (hold) | Switch between wireframe, solid, rendered, material views |
| Shift ` | Enter walk/fly mode (like UE5) | 
| Ctrl Shift ` | Toggle on/off 3D axis overlays |
