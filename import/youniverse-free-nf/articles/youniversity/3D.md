# 3D: Characters process pipleine
The complete general process for creating characters in 3D

This process is split into two parts: creating a base human model, and using that model, to create personalized characters

1. Vision
2. Sculpting
3. Retopology
4. UV Unwrapping
5. Texturing
6. Rigging
7. Animating


# Detailed process


1. Character
   1. Vision: Inspiration, material, references. Starting with something that inspires you. Wether it's a specific emotion you want to evoke
   2. Artstyle: this has a huge impact in every step of the process. Confining oneself to a specific artstyle can feel limiting, but it's within those limits that will allow you to focus on more important, organic parts of the character. [Read more here](artstyles.md)
   3. Concept: Sketching, ideas, concepts: This is the stage of testing out different ideas before commiting to them.
2. Base model
   1. Sculpting:
   2. Retopology
   3. UV Unwrapping
   4. Texturing
   5. Rigging
      1. Weight painting   
   6. Animation
3. Accessories
   1. Applyinig accessories, clothing & more to an pre-existing base model. Ideally, this model is already rigged, and the clothing can simply be weight-painted onto the rig.
