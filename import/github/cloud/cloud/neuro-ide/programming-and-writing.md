# Why use programming with note-taking?

## 1.0 Consistency
Consistency, standardization and human input reduction is the main purpose of programming. When combining programming with note-taking, the possibilities are endless. And thus, require consistent constraint in order to be useful in everyday usage.

## 2.0 Abstraction -The reduction of human input, but not reduction of human agency.
Abstraction can be lossely defined as "The reduction of human input, but not reduction of human agency."

The SQL query language is a great example of this. It provides a micro-programming-language (MPL)
