In order for... ahem... well... order to arise... there has to be a single starting place. Essentially, it's like the Big-Bang theory... except it doesn't assume creation... it's about the fact that creation itself cannot exist without a starting place.

This helps us understand how deeply complex systems came to be. How did AI systems learn to generate images from text? How do computers process software and human input? 

It all starts from an origin point.

The deeper, more important part of this idea is that the origin point is never perfect, becuase it needs to be messy. It needs to be experimental, and in the moment. Without that, there is no movement, and no decision. There is such a thing as perfection, but we as human beings are too limited by time to wait for things to be ideal.
