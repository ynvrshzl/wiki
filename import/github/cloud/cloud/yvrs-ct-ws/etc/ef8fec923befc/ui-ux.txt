UI UX

This is a very challenging field. Not becusae of it's complexity, but becuase most of UI/UX design is psychology and art. The biggest challenge in all of UI/UX design, isn't how to make something look good, but it's understanding what makes something feel good. More importantly, how can we undersatnd how we human beings visually see the world through, and how can we map that into the digital world as interffaces?

The interface is esentially a translation layer between the human being, and the machine. The UI is one of many examples of interfaces. A button could be an interface. 

The fundamental building blocks of UI/UX design could be described as:
- Shape: Accesibilty
- Color: Expressiveness
- Imagination: Capture
- Structure: Readablity
- Harmony: Cohesivenesss
