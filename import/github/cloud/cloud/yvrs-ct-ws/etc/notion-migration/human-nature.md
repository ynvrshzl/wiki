# What is wrong with us?

# Fundamental human nature
A human being must be in the place they belong, and given what they need not only to function but to grow. It's unfortunate that the world we live in has made life more convenient and overall, safer, but we've sacrificed what we human beings fundamentally need: belonging.

There is more media in the world **"social media"** than actual socializing.
