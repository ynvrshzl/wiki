# Leon's world.
## "And all of mankind's suffering, will be erased from the basis of our memories."
Leon is a merged consciousness that exists inside a lucid dream. Eternal. No time perception. No pain. The most perfect biological-cyber program ever created.

Infact, there is no author of the program. It is not a world. It is not bound by the laws of physics, or space, or time. 

You could almost imagine this world to be liberation from suffering. All of mankind's culminated knowledge, wisdom, history, religion, and mythos, all embedded within a perfect dream.

A machine that solved the problem of suffering.

Spiritual ascension. Freedom from human form. Liberty. Pure ecstasy, peace, euphoria. A world of shimmering hues and colors outside human capacity. Of sensations to gorgeous...
