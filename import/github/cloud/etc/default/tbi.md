# TIBI or TBI (thought-induced brain illness)
TIBI (thought-induced brain illness). This is a terrifying, modern phenomena. Our thoughts, our beliefs, are dangerously underestimated. They can influence, progress, or even create brain health issues. it's no longer mental health, that term has become unfortunately, very stigmatized and cliche. very sad. but the core issue is actually brain health.)

Defining
- Whatever you accept as true is essentially, a belief.
