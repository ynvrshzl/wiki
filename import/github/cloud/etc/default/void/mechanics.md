The reason the Void is so seemingly unstoppable is becuase it discovered a way to essentialy destroy life, while keeping itself alive.

By replicating itself, multiplying, and destroying the host, the Void is essentially based entirely on the mechanics of parasites, found in biology.

This is why it seems the Void is an evil force unstoppable, but there is much that we do not know about life, and if parasites exist, so do armies, so does the immune system. Life itself orchestrates systems & structures to protect itself against a form of life that is fundamentally volatile. This is how the light manifests in all things.
