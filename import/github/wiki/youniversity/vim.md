# Learning Vim
Ready to be a wizard on the keys? Here, we are learning together, Vim, not the editor, but the hotkeys.

## Is it worth it learning vim?
Do you type alot? Or maybe you work alot with the keyboard? Vim actually doesn't just save time, it shapes how you think about working with text.

For example, imagine trying to jump the text cursor at a specific letter in the document. That's really annoying to do with a mouse, but with Vim, it's as simple as staying on your keyboard!

## 1.0 Motion
"The fundamentals"

Vim is all about learning building blocks and learning to combine them into a sensory language. This is essentialy what  hotkeys are, vim is like a nerdy langauge.
