
## Components 
Components of base core programme
### Core
- Cross-platform 
- Offline/Online mode
- Built-in FTP or Localhost server 
- STD File-system 
- Markdown editor 
- Commands
- Basic UI 

### Hypercore 
- Application is entirely editable (using JS)
- [SQL](https://sql.org) inspired micro programming language to assemble application, and provide programming capabilities to note-taking.
- In-app webview (like obsidian web core-plugin)
- Domain
