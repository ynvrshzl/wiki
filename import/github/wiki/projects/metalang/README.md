## Linear path traversal (LPT) - The only sustainable engineering process for humans
Essentially the only manageable way we can beings navigate complex systems, is by walking linear paths. And sort of building memory as we go. That's how we build a cognitive map. 

If you think about it, fundamentally, all of our architectures, modules, classes, and techniques, are forms of LPT what we're doing in code, only there really is no orientation in the design, I think that's actually the primary source of all the overwhelm we experience. because as soon as we enter like a cognitive map and we understand conceptual models we can translate that into programming terms and that's essentially what programming is!

## Systems emerge from organic LPT
