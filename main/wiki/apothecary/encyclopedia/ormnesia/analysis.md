# Analysis of Oramnesia — Memory Loss of meaning 
**Why does the human being forget important things?** 

Abstract: we were noticing that there is a recurring pattern in our past, it is possibly a common pattern for all trauma survivors. it almost seems to be an amnesia... where the individual forgets the path they are on, or what was once meaningful to them. we gave this a name, [Oramnesia](oramnesia)

Personal synopsis: 
- Routine, Rhythm and Cycles are the core foundation of memory.
- Meaning has an exhaustive half-life.
- if you do not surround yourself with sources of meaning everyday, it is not part of your life, if it is not embodied, then by definition, no access...
