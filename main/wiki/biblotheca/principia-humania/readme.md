
# Principia Humana
a field guide for life-threatening response 

##  Abstract
this document contains everything a human being should know, in case of life threatening situations. this really should not be survival guide level theory, this needs to be taught to every human being, specifically in our schools. but political issues aside, this document is designed to be portable enough to hold within human working memory.

## S Structure
it's important that we structure this document to me as accessible and practical as possible. the information shared here should be portable, so you can take it with you, should  the unfortunate situation ever arise. 

the design of this document, is information bubbles. meaning, each topic is collapsible into a single bubble, should a situation arise. which is essentially how human working memory is structured. and so now that we understand the mechanics of human memory, we can structure our documents to increase the likelihood that this will be helpful. 


## T topics
current bubbles include:
- life resuscitation:  how to perform CPR 
- someone is choking, how to perform heimlich maneuver
- how to protect yourself against a wild dog 
- how to identify rabies
- how to disarm someone with a knife 
- how to safely escape fire
- how to help someone who is suicidal 
- if you are choking, how to perform heimlich maneuver on yourself

some more conceptual side notes, which are abstracts and most likely not part of the core document, include:
- always have a Swiss army knife with you.
- never trust a stranger online 
- never click random links on the internet.
