# Core Requirements

This program is a mixture of sandbox play, and solving problems in our life. essentially we have a very noisy brain, and I've learned not to capture everything, the human mind has its own built-in organic garbage collection and filtering.

but the ability to create your own wiki, that's essentially how we learned to use obsidian. we're currently writing these documents inside of notion, just giving it a little try while we experiment with lots of tools. I do notice that there is a increasing anxiety, experimenting too much with our systems, and almost feels like there's no single source, no place to call home digitally, and perhaps this program is a way of providing a digital home built entirely out of plain text files!

if you think about it, the web is organized by HTML files, links that connect to them, and systems that make a web pages interactive. that's essentially the concept behind this neuro-ide. using the browser as a web technology runtime.
