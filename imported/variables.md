A variable is memory. 
A variable is a fundamental unit of time encoded as space.

This may be obvious today, but specifically; a variable = memory, and memory = variable.

What does this mean? 

- Variables store information as memory.
- Memory are patterns of time, encoded in space; time encoded in space.

Why do we use variables? 
The problem a variable solves is two-fold:
- A variable encodes patterns, essentially solving the problem of repeating the same information within a system.
- A variable becomes a link, restore point, a shapshot, a state.
