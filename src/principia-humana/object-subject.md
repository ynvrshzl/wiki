# Object vs Subject
When we process the real world, we are constantly analyzing the boundaries between; __Object and Subject.__

__What is Objective? vs. What is Subjective?__

Two completely different individuals, are not merely defined by external, extrinsic conditions; we human beings are defined by;
- Intrinsic: How we each individually process the same shared world
- Extrinsic: How we each individually act and behave
